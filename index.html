What's Banana.Core?

Banana.Core is a high level library for managing iOS device written in C# using libimobiledevice instead of Apple's MobileDevice.dll

Code friendly!

Made for beeing easy for developers. Made for beeing sexy for users.

Open source!

Do you want to build your own library? Then download the source code and pull here your updates! :)
